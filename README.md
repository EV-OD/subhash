# Adv. Subhash Lamichhane - Legal Portfolio & Resource Center

A comprehensive professional portfolio and legal resource platform for **Adv. Subhash Lamichhane**, a licensed advocate (Top 10 in the 33rd Bar Council Exam) and legal researcher based in Nepal.

This website serves as a hub for legal professionals, students, and the general public to access legal articles, research papers, case laws, and academic resources focused on Nepali and International Law.

## Key Features

- **Legal Articles & Research Papers:** In-depth analysis of constitutional doctrines, election processes, and statutory frameworks (e.g., Human Trafficking and Transportation (Control) Act).
- **Case Law Database:** Curated selection of significant case laws and legal precedents.
- **Academic Resources:** Resources for law students and practitioners, including academia and practice areas.
- **Practice Areas:** Detailed information on expertise in Corporate Law, Criminal Law, Civil Law, and more.
- **Video Library:** Educational and informative videos on legal topics.
- **Modern UI/UX:** Built with a clean, professional aesthetic using Tailwind CSS and Radix UI components.
- **Dynamic Content Management:** Powered by **Keystatic CMS** for easy updates to articles, research, and profile information.
- **Interactive Features:** View counters (Upstash Redis), Disqus comments, and social sharing.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router, TypeScript)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **CMS:** [Keystatic](https://keystatic.com/) (Local/GitHub-based)
- **UI Components:** [Radix UI](https://www.radix-ui.com/), [Lucide React](https://lucide.dev/), [Embla Carousel](https://www.embla-carousel.com/)
- **State/Data:** [Upstash Redis](https://upstash.com/) (View counters), [Firebase](https://firebase.google.com/)
- **PDF Viewer:** [@embedpdf/react-pdf-viewer](https://www.npmjs.com/package/@embedpdf/react-pdf-viewer)
- **AI Integration:** [Google Genkit](https://firebase.google.com/docs/genkit) (for specialized legal AI features)

## 🏁 Getting Started

### Prerequisites

- Node.js (v18 or later recommended)
- `npm` or `pnpm`

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/subhash-portfolio.git
    cd subhash-portfolio
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Set up environment variables:
    Create a `.env` file based on the environment requirements (Upstash, Firebase, etc.).

4.  Run the development server:
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:9003`.

### CMS Access

You can manage content locally by visiting `/keystatic` in your browser while the development server is running.
